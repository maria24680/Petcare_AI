# 🐾 PetCare AI – Smart Pet Care Assistant (Frontend)

PetCare AI is a modern, responsive, AI-powered pet care web application built with **Next.js**, **TypeScript**, and **Tailwind CSS**. It helps pet owners manage their pets, receive AI-powered pet care guidance, and get personalized food recommendations through an intuitive and user-friendly interface.

# 🤖 AI Features

### 🩺 AI Pet Care Assistant

Ask pet-related questions and receive AI-generated responses.

Examples:

- My dog is losing hair.
- My cat isn't eating.
- My puppy has itchy skin.

Features:

- Conversation History
- Regenerate Response
- Loading Indicator
- Smart Suggestions

---

### 🥩 AI Food Recommendation

Generate personalized food plans based on:

# 📦 Installation

Clone the repository

```bash
git clone https://github.com/your-username/petcare-ai-client.git
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open

```
http://localhost:3000
```

---

# ⚙️ Environment Variables

Create a `.env.local` file and add:

```env
NEXT_PUBLIC_API_URL=
NEXT_PUBLIC_APP_URL=
NEXT_PUBLIC_BETTER_AUTH_URL=
NEXT_PUBLIC_GOOGLE_CLIENT_ID=
NEXT_PUBLIC_GEMINI_API_KEY=
```
